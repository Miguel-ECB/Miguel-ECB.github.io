Procesadores AMD

Gama Baja

AMD EPYC 3000 Series:

  • Núcleos/Hilos: 4 núcleos / 8 hilos.
  • Frecuencia base: 2.8 GHz.
  • Caché: 8 MB L3.
  • TDP: 35W.
  • Uso recomendado: Servidores de entrada, almacenamiento en la nube básica y aplicaciones de red.

Gama Media

AMD EPYC 7300 Series:

  • Núcleos/Hilos: 16 núcleos / 32 hilos.
  • Frecuencia base: 3.0 GHz, Boost hasta 3.8 GHz.
  • Caché: 64 MB L3.
  • TDP: 120W.
  • Uso recomendado: Virtualización, cargas de trabajo de bases de datos y servidores web medianos.

Gama Alta

AMD EPYC 9000 Series:

  • Núcleos/Hilos: 96 núcleos / 192 hilos.
  • Frecuencia base: 2.7 GHz, Boost hasta 4.0 GHz.
  • Caché: 256 MB L3.
  • TDP: 320W.
  • Uso recomendado: Centros de datos de alto rendimiento, inteligencia artificial, renderizado y análisis de big data.
